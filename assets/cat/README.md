# Cat Corner

The customer display now uses an **inline SVG tuxedo cat** built into `display.html` directly — no external animation files needed.

The cat cycles through 4 states every 14 seconds: walking → sleeping → playing → eating. All animations are pure CSS keyframes (no JS animation library, no Lottie, no dependencies).

This folder is reserved in case we want to swap in custom Lottie animations later. For now, leave it empty.
